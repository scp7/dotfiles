# WezTerm Cheat Sheet

## Tabs

| Keys | Action |
|------|--------|
| `Cmd + T` | New tab |
| `Cmd + W` | Close tab |
| `Ctrl + Tab` | Next tab |
| `Ctrl + Shift + Tab` | Previous tab |
| `Cmd + 1-9` | Go to tab N |

## Panes (splits)

| Keys | Action |
|------|--------|
| `Cmd + D` | Split right |
| `Cmd + Shift + D` | Split down |
| `Cmd + Shift + Left` | Activate pane left |
| `Cmd + Shift + Right` | Activate pane right |
| `Cmd + Shift + Up` | Activate pane up |
| `Cmd + Shift + Down` | Activate pane down |
| `Cmd + W` | Close pane |
| `Cmd + Shift + Z` | Toggle pane zoom |

## Font / Window

| Keys | Action |
|------|--------|
| `Cmd + =` | Increase font size |
| `Cmd + -` | Decrease font size |
| `Cmd + 0` | Reset font size |
| `Cmd + Enter` | Toggle fullscreen |
| `Cmd + N` | New window |

## Copy / Search

| Keys | Action |
|------|--------|
| `Cmd + C` | Copy selection with confirmation |
| `Ctrl + Shift + C` | Copy selection with confirmation |
| `Cmd + V` | Paste |
| `Cmd + F` | Search |
| `Cmd + Shift + Space` | Quick select mode |
| `Cmd + K` | Clear scrollback |

## Copy Mode

Use this only when you want to copy without the mouse.

| Keys | Action |
|------|--------|
| `Ctrl + Shift + X` | Enter copy mode |
| `Space` | Start selecting |
| `Arrow keys` / `hjkl` | Move selection |
| `y` / `Enter` | Copy and exit |
| `Esc` / `q` | Exit copy mode |

## Scrollback

| Keys | Action |
|------|--------|
| `Shift + PageUp` | Scroll up |
| `Shift + PageDown` | Scroll down |

## Custom (from config)

| Keys | Action |
|------|--------|
| `Ctrl + Enter` | Send Ctrl+Enter to shell |
| `Shift + Enter` | Send Shift+Enter to shell |

## Config

| Item | Value |
|------|-------|
| Font | JetBrainsMono Nerd Font (16pt) |
| Color scheme | Kanagawa (Gogh) |
| Opacity | 0.94 with background blur |
| Tab bar | Disabled |
| Startup | Maximized window |
